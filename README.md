# Creating a "create-react-app" with Docker Example

## Quickstart

With docker-compose:

```
docker-compose up (-d)
```

or you can build and run using Docker:

```
docker build -t docker-react .
docker run -i docker-react
```

Access test page using address:

```
http://localhost:8080/
```
